## 프로젝트 개요

- **정체**: 브라우저에서 바로 열리는 단일 파일 웹툰 컷 배치 에디터. 서버·빌드·설치 불필요.
- **핵심 파일**: `index.html` 하나에 HTML+CSS+Vanilla JS 전부 포함. 외부 의존성은 PNG 저장용 html2canvas(CDN)뿐.
- **배포**: GitHub 저장소 `github.com/kanton7/webtoon-editor`. 루트의 `index.html`이 그대로 동작.

## 구현된 기능

- **재귀 분할 레이아웃**: 컷을 상·하(`Shift`+클릭) / 좌·우(`Ctrl`+클릭)로 무한 중첩 분할. 재귀 트리(`leaf`/`h`/`v`) + Flexbox.
- **인터랙티브 리사이즈**: 경계선 드래그로 옆 컷과 크기 분배. `Shift` 미세조정, `Ctrl` 이미지 크기 고정(크롭 모드).
- **이미지 넣기**: 더블클릭 또는 드래그&드롭. 선택 후 드래그로 위치 이동, `+`/`−`로 확대·축소.
- **채움/맞춤/고정**: 컷 크기에 맞춰 스케일하거나 이미지 원본 크기를 고정한 채 프레임만 크롭.
- **사선 컷**: 꼭짓점 드래그로 대각선 경계 생성. 맞닿은 옆 컷이 경계선 공유(옆칸 침범), 인접 꼭짓점 자동 스냅(`Alt` 해제). CSS clip-path + SVG 외곽선 테두리.
- **컷 이동·합치기**: 컷을 다른 컷 가장자리로 드래그해 재배치/합치기.
- **컷 간격(거터)·테두리**: 두께·색 지정. 사선은 SVG polygon으로 정확히 렌더.
- **말풍선 25종**: 팔레트에서 드래그해 배치. 글자 입력(더블클릭), 이동, 크기 조절, **회전(초록 핸들)**, **좌우 반전(⇄ 버튼, 꼬리 방향)**. 종류: 말하기·감정·강조·생각특수·효과·나레이션 6그룹.
- **실행취소/다시실행**: `Ctrl+Z` / `Ctrl+Shift+Z`. 기록 횟수 조정(기본 5).
- **원고 규격 프리셋**: 네이버 690 / 카카오 720 / 레진 800 / 직접 입력.
- **PNG 내보내기**: 전체 화면을 2배 해상도 한 장으로 저장(html2canvas).

## 코드 구조 메모 (index.html 내부)

- **데이터 모델**: `state.root` = 재귀 트리. 노드 `{t:'leaf'|'h'|'v', w, kids...}`. leaf에 `img/color/fit/pos/scale/fixed/poly`. 말풍선은 `state.balloons[]`(`type/x/y/w/h/text/rot/flip`).
- **렌더**: `render()` → `buildInto`/`buildNode`/`buildLeaf`로 DOM 재생성 + `renderBalloons()`. 상태 바뀔 때마다 전체 재렌더 방식.
- **히스토리**: `snap()`으로 깊은 복사, `pushHistory()` 후 변경. undo/redo 스택.
- **사선**: `startCorner` + `neighborInfo`로 이웃 컷 꼭짓점 동기화.
- **말풍선 모양**: `balloonInner(type)`가 SVG path 문자열 생성(별/구름/불꽃/폭발/집중선 등 함수형 지오메트리). `balloonSVG()`로 감쌈.
- **초기화**(파일 맨 끝): `render(); buildPalette(); syncInputs(); updateUndoUI();` + 전역 keydown 리스너.

### 주의: 과거 버그 이력
- 한때 파일이 `exportPNG` 마지막 줄에서 **잘려서**(truncate) 초기화 코드와 단축키가 통째로 사라져 화면·팔레트가 안 뜬 적 있음. 저장 후엔 파일 끝이 `</script></body></html>`로 온전히 닫혔는지, 초기화 4줄이 있는지 확인할 것.
