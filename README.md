# 🎬 웹툰 컷 에디터 (Webtoon Panel Editor)

브라우저에서 바로 열리는 **단일 HTML 웹툰 컷 배치 에디터**입니다. 설치·서버·빌드 없이 파일 하나로 동작합니다. 이미지를 컷에 채우고, 컷을 자유롭게 분할·리사이즈·사선 처리하여 하나의 웹툰 원고 이미지로 내보낼 수 있습니다.

> **by kanton7** · ver. 0.12 · 📧 kanton7@naver.com

---

## ✨ 주요 기능

- **재귀 분할 레이아웃** — 컷을 상·하 / 좌·우로 무한 중첩 분할. 한 컷만 나누고 옆 컷은 전체 높이를 유지합니다.
- **인터랙티브 리사이즈** — 컷 사이 경계선을 드래그하면 옆 컷이 자동으로 크기를 나눠 가집니다. `Shift`로 미세 조절.
- **이미지 넣기·조정** — 더블클릭 또는 드래그&드롭으로 이미지 삽입. 선택 후 드래그로 위치 이동, `+`/`−`로 확대·축소.
- **채움 / 맞춤 / 고정** — 컷 크기에 맞춰 이미지를 스케일하거나(맞춤·채움), 크기를 고정한 채 프레임만 크롭(`Ctrl`+경계선 드래그).
- **사선 컷** — 꼭짓점을 드래그해 대각선 경계를 만들고, 맞닿은 옆 컷이 경계선을 함께 공유합니다. 인접 꼭짓점에 자동 스냅.
- **컷 이동·합치기** — 컷을 다른 컷의 상·하·좌·우 가장자리로 드래그해 재배치하거나 같은 줄로 합칩니다.
- **컷 간격 · 테두리** — 컷 사이 간격(거터)과 테두리 두께·색상 지정(사선 테두리는 SVG 외곽선으로 정확히 렌더링).
- **실행취소 / 다시실행** — `Ctrl+Z` / `Ctrl+Shift+Z`. 기록 횟수 조정 가능(기본 5회).
- **원고 규격 프리셋** — 네이버 690 · 카카오페이지 720 · 레진 800 · 직접 입력.
- **PNG 내보내기** — 배치한 전체 화면을 2배 해상도 이미지 한 장으로 저장.

## 🚀 사용 방법

`webtoon-editor.html` 파일을 브라우저(크롬·엣지·사파리 등)에서 열면 바로 사용할 수 있습니다.

### 단축키 / 조작

| 조작 | 동작 |
|---|---|
| 더블클릭 | 컷에 이미지 넣기 |
| 클릭 | 컷 선택 (꼭짓점 표시) |
| `Shift` + 클릭 | 상·하 분할 |
| `Ctrl` + 클릭 | 좌·우 분할 |
| 경계선 드래그 | 크기 조절 (`Shift` 미세 · `Ctrl` 이미지 고정 크롭) |
| 이미지 드래그 | 선택 후 위치 이동 |
| `+` / `−` | 선택 컷 이미지 확대·축소 |
| 꼭짓점 드래그 | 사선 만들기 (`Shift` 수평·수직 · `Alt` 스냅 해제 · 더블클릭 해제) |
| 컷 드래그 | 다른 컷 가장자리로 이동·합치기 |
| `Ctrl` + `Z` / `Ctrl` + `Shift` + `Z` | 되돌리기 / 다시실행 |

## 🌐 배포하기

이 프로젝트는 정적 파일 하나이므로 어디에 올리든 그대로 동작합니다. 배포 시 **파일 이름을 `index.html`로 바꾸면** 루트 주소에서 바로 열립니다.

### GitHub Pages

1. GitHub에서 새 저장소(repository)를 만듭니다.
2. `webtoon-editor.html`을 `index.html`로 이름을 바꿔 업로드하고, 이 `README.md`도 함께 커밋합니다.
3. 저장소 **Settings → Pages** 로 이동합니다.
4. *Source*를 `main` 브랜치 / `/root`로 지정하고 저장합니다.
5. 잠시 후 `https://<사용자명>.github.io/<저장소명>/` 주소로 공개됩니다.

### Vercel

1. [vercel.com](https://vercel.com) 에 GitHub 계정으로 로그인합니다.
2. **Add New → Project** 에서 위 GitHub 저장소를 가져옵니다(Import).
3. 프레임워크 설정은 **Other**(정적 사이트)로 두고 별도 빌드 명령 없이 **Deploy**를 누릅니다.
4. 배포가 끝나면 `https://<프로젝트명>.vercel.app` 주소가 생성됩니다.

> 별도 빌드 과정이 없으므로, 저장소에 `index.html`만 있으면 GitHub·Vercel 모두 자동으로 인식합니다.

## 🛠 기술 노트

- 순수 **HTML + CSS + Vanilla JavaScript** 단일 파일. 프레임워크·빌드 도구 불필요.
- 레이아웃은 재귀 트리(`leaf` / `h`(가로) / `v`(세로) 분할) + CSS Flexbox로 구성.
- 사선 컷은 CSS `clip-path`, 사선 테두리는 인라인 SVG `polygon` 외곽선으로 렌더링.
- PNG 내보내기는 [html2canvas](https://html2canvas.hertzen.com/)를 CDN에서 로드합니다. **편집은 오프라인에서도 되지만, PNG 저장 기능만 인터넷 연결이 필요**합니다.
