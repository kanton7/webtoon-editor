# 🎬 웹툰 컷 에디터 (Webtoon Panel Editor)

브라우저에서 바로 열리는 **단일 HTML 웹툰 컷 배치 에디터**입니다. 설치·서버·빌드 없이 파일 하나로 동작합니다. 이미지를 컷에 채우고, 컷을 자유롭게 분할·리사이즈·사선 처리하고, 말풍선을 올려 하나의 웹툰 원고 이미지로 내보낼 수 있습니다. 작업 중인 페이지는 파일로 저장해 나중에 이어서 작업할 수 있습니다.

## ✨ 주요 기능

### 컷 레이아웃
- **재귀 분할 레이아웃** — 컷을 상·하 / 좌·우로 무한 중첩 분할. 한 컷만 나누고 옆 컷은 전체 높이를 유지합니다.
- **인터랙티브 리사이즈** — 컷 사이 경계선을 드래그하면 옆 컷이 자동으로 크기를 나눠 가집니다. `Shift`로 미세 조절.
- **사선 컷** — 꼭짓점을 드래그해 대각선 경계를 만들고, 맞닿은 옆 컷이 경계선을 함께 공유합니다. 인접 꼭짓점에 자동 스냅.
- **컷 이동·합치기** — 컷을 다른 컷의 상·하·좌·우 가장자리로 드래그해 재배치하거나 같은 줄로 합칩니다.
- **컷 간격 · 테두리** — 컷 사이 간격(거터)과 테두리 두께·색상 지정(사선 테두리는 SVG 외곽선으로 정확히 렌더링).
- **🔒 컷 잠금** — 잠그면 뒤 컷·이미지가 선택·이동되지 않아, 말풍선 작업 중 배경이 실수로 움직이는 것을 막아줍니다.

### 이미지 넣기·조정
- **이미지 삽입** — 컷 더블클릭, 드래그&드롭, 또는 좌측 보관함 썸네일을 컷 위로 드래그해 삽입.
- **채움 / 맞춤 / 고정** — 컷 크기에 맞춰 이미지를 스케일하거나(맞춤·채움), 크기를 고정한 채 프레임만 크롭(`Ctrl`+경계선 드래그).
- **위치 조정** — 컷 선택 후 이미지를 드래그해 위치 이동. **채움 모드에서는 이미지가 컷을 꽉 채운 채 보이는 부분만 이동(여백·잘림 없음)**, `+`/`−`로 확대·축소.

### 🖼️ 이미지 보관함 (좌측 패널)
- **폴더 열기 / 이미지 추가** — 폴더를 통째로 불러오거나 개별 이미지를 추가해 썸네일로 보관.
- **폴더별 그룹 · 탭** — 불러온 폴더 이름으로 자동 그룹화. 상단 탭으로 폴더별 전환(장수 표시). 이미지가 많아도 정리해서 볼 수 있습니다.
- **중복 방지** — 이미 불러온 이미지(파일명+크기 동일)는 다시 불러오지 않습니다.
- **폴더 관리** — 새 폴더 생성, 폴더 이름 변경, 폴더 삭제.
- **다중 선택 → 폴더 이동** — 썸네일 체크박스로 여러 장을 골라 원하는 폴더로 한꺼번에 이동. "현재목록 전체선택" 지원.
- **삽입** — 썸네일을 컷 위로 드래그하거나, 컷 선택 후 썸네일을 더블클릭해 넣기.

### 💬 말풍선
- **말풍선 템플릿** — 25종(기본·속삭임·외침·비명·생각·폭발·전자음·플래시·집중선·나레이션 등)을 팔레트에서 드래그해 배치.
- **편집** — 더블클릭 글자 입력, 드래그 이동, **8방향 핸들 크기 조절**, 위쪽 초록 핸들로 **회전**(`Shift` 15° 스냅), 좌우 반전.
- **✏️ 말풍선 직접 그리기** — 캔버스에 드래그해 타원 크기를 그리고, 꼬리 끝을 클릭하면 그 방향으로 **얇은 곡선 꼬리**가 달린 말풍선이 생성됩니다(`Esc`=꼬리 없이/취소).
- **꼬리 조정** — 생성 후 채운 주황 핸들(꼬리 끝)·빈 주황 핸들(꼬리 시작점)을 드래그해 방향·길이·붙는 위치를 바꾸고, 끝 핸들 더블클릭으로 꼬리 제거. 꼬리는 중심 기준 위치에 따라 좌/우로 자연스럽게 휩니다.
- **테두리 굵기** — 선택한 말풍선(또는 새 말풍선 기본값)의 외곽선 굵기를 **px 단위**로 조절(px 입력·슬라이더·`−`/`＋` 버튼·키보드 `+`/`−`).

### 저장·내보내기
- **💾 페이지 저장 / 📂 불러오기** — 현재 페이지 전체(컷 배치·삽입 이미지·말풍선·설정·보관함·폴더 구성)를 `.json` 파일로 저장하고, 다시 불러와 이어서 작업. 이미지가 파일에 포함돼 다른 컴퓨터에서 열어도 유지됩니다.
- **원고 규격 프리셋** — 네이버 690 · 카카오페이지 720 · 레진 800 · 직접 입력.
- **PNG 내보내기** — 배치한 전체 화면을 2배 해상도 이미지 한 장으로 저장.
- **실행취소 / 다시실행** — `Ctrl+Z` / `Ctrl+Shift+Z`. 기록 횟수 조정 가능(기본 5회).

## 🚀 사용 방법

`index.html` 파일을 브라우저(크롬·엣지·사파리 등)에서 열면 바로 사용할 수 있습니다.

### 단축키 / 조작

| 조작 | 동작 |
|---|---|
| 더블클릭 | 컷에 이미지 넣기 |
| 클릭 | 컷 선택 (꼭짓점 표시) |
| `Shift` + 클릭 | 상·하 분할 |
| `Ctrl` + 클릭 | 좌·우 분할 |
| 경계선 드래그 | 크기 조절 (`Shift` 미세 · `Ctrl` 이미지 고정 크롭) |
| 이미지 드래그 | 선택 후 위치 이동 (채움 모드=여백 없이 보이는 영역 이동) |
| `+` / `−` (컷 선택 시) | 이미지 확대·축소 |
| 꼭짓점 드래그 | 사선 만들기 (`Shift` 수평·수직 · `Alt` 스냅 해제 · 더블클릭 해제) |
| 컷 드래그 | 다른 컷 가장자리로 이동·합치기 |
| 말풍선 클릭 | 선택 (8방향 크기 핸들 · 회전 핸들 표시) |
| 말풍선 더블클릭 | 글자 입력 |
| 회전 핸들 드래그 | 회전 (`Shift` 15° 스냅) |
| `+` / `−` (말풍선 선택 시) | 테두리 굵기 조절 |
| `Ctrl` + `Z` / `Ctrl` + `Shift` + `Z` | 되돌리기 / 다시실행 |

> 좌측 상단 **보관함 / 사용법** 탭으로 이미지 보관함과 조작 안내를 전환할 수 있습니다.

## 🌐 배포하기

이 프로젝트는 정적 파일 하나(`index.html`)이므로 어디에 올리든 그대로 동작합니다. 루트의 `index.html`을 자동으로 인식합니다.

### GitHub Pages

1. GitHub에서 새 저장소(repository)를 만듭니다.
2. `index.html`과 이 `README.md`를 업로드하고 커밋합니다.
3. 저장소 **Settings → Pages** 로 이동합니다.
4. *Source*를 `main` 브랜치 / `/root`로 지정하고 저장합니다.
5. 잠시 후 `https://<사용자명>.github.io/<저장소명>/` 주소로 공개됩니다.

### Vercel

1. [vercel.com](https://vercel.com) 에 GitHub 계정으로 로그인합니다.
2. **Add New → Project** 에서 위 GitHub 저장소를 가져옵니다(Import).
3. 프레임워크 설정은 **Other**(정적 사이트)로 두고 별도 빌드 명령 없이 **Deploy**를 누릅니다.
4. 배포가 끝나면 `https://<프로젝트명>.vercel.app` 주소가 생성됩니다.

> 별도 빌드 과정이 없으므로, 저장소에 `index.html`만 있으면 GitHub·Vercel 모두 자동으로 인식합니다.

## 🛠 기술 노트

- 순수 **HTML + CSS + Vanilla JavaScript** 단일 파일. 프레임워크·빌드 도구 불필요.
- 레이아웃은 재귀 트리(`leaf` / `h`(가로) / `v`(세로) 분할) + CSS Flexbox로 구성.
- 사선 컷은 CSS `clip-path`, 사선 테두리는 인라인 SVG `polygon` 외곽선으로 렌더링.
- 말풍선은 SVG 패스로 생성. 직접 그린 말풍선의 꼬리는 타원 외곽선과 하나로 이어지는 베지어 곡선으로, 클릭 지점을 향하도록 계산됩니다. 테두리 굵기는 배수로 관리해 얇은 디테일 선의 비율을 유지합니다.
- 이미지·페이지 저장은 브라우저 안에서 처리되며, 저장 파일(`.json`)에 이미지가 Data URL로 포함됩니다.
- 폴더 불러오기는 브라우저 보안상 선택한 시점의 파일들을 읽어옵니다(실시간 폴더 동기화는 아님).
- PNG 내보내기는 [html2canvas](https://html2canvas.hertzen.com/)를 CDN에서 로드합니다. **편집은 오프라인에서도 되지만, PNG 저장 기능만 인터넷 연결이 필요**합니다.
